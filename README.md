#  World Happiness Report 2019 — EDA

## Project Overview
Exploratory Data Analysis on the World Happiness Report 2019 dataset
covering 156 countries. Built as part of the Pluto Academy AI & ML Internship.

##  What This Project Covers
- Data loading, inspection and cleaning
- 5 EDA questions answered with Pandas
- 6 visualizations (bar, scatter, histogram, heatmap, line, pie)
- 5 business insights based on findings

##  Tools Used
Python | Pandas | NumPy | Matplotlib | Seaborn | Google Colab

##  Dataset
World Happiness Report 2019 — sourced from Kaggle
https://www.kaggle.com/datasets/unsdsn/world-happiness

##  Key Findings
- Nordic countries consistently dominate happiness rankings
- GDP per capita is the strongest predictor of happiness (r = 0.79)
- A 2.89 point happiness gap exists between wealthy and poor nations
- Generosity has almost no correlation with happiness (r = 0.076)
- Most countries cluster around the global average of 5.41
